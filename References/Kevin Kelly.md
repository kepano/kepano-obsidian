## Books

```dataview
table without id
	file.link as Title,
	year as Year,
	rating as Rating
where
	contains(category,[[Books]]) and
	contains(author,this.file.link)
sort rating desc
```

# Clippings

```dataview
table without id
	file.link as Title,
	published as Published
where
	contains(category,[[Clippings]]) and
	contains(author,this.file.link)
sort rating desc
```

# Podcast episodes

